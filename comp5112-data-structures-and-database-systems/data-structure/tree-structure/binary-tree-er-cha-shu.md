# Binary Tree（二叉树）

二叉树是一个ordered tree，the left subtree is before the right subtree。

## Node Attributes

* v.element： data element
* v.left：左子节点
* v.right：右子节点
* v.p：父节点

## Inorder Traversal（中序遍历）

visit a node after its left subtree and before its right subtree
